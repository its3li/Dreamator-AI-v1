# Dreamator AI

An AI-powered image generation application built with React and Vite.

## Features
- AI Image Generation
- Image Download
- Share Functionality
- Bilingual Support (English/Arabic)

## Tech Stack
- React
- Vite

- Framer Motion
- Vercel Analytics
